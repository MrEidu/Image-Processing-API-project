Api Processing image home page

Welcome to my Udacity Project.

This project support opening and resizing images, as well as deleting and uploading them.

For resizing images

Parameters: /resize?file=fileName.format&width=number

Try extending the url after "/api" with the following lines:

  • api/resize?file=015.jpg&width=500&height=100
  • api/resize?file=20211219_141722%201.jpg&width=200

For uploading images

Add api/upload to the url (not implemented)

For deleting images

Add api/delete to the url.

Parameters: api/delete/delete?file=fileName.format
